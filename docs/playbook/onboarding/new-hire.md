### Day 1: Welcome & Setup

**Morning (9:00 AM - 12:00 PM)**
- [ ] Welcome breakfast with team
- [ ] Meet your manager and set goals
- [ ] Get laptop and accounts setup
- [ ] Set up GitHub access (duet-company org)

- [ ] Review company vision and strategy (see [vision repo](https://github.com/duet-company/vision))
- [ ] Read architecture documentation
- [ ] Set up development environment (see [Developer Setup](./developer-setup.md))

- [ ] Say hello in team chat

### Day 2: Product Deep Dive

**Morning**
- [ ] Product walkthrough with PM/Engineering
- [ ] Demo of AI agents (Query, Designer, Support, Ops)
- [ ] Review current roadmap and OKRs
- [ ] Understand our tech stack: ClickHouse, Kubernetes, FastAPI, React

**Afternoon**
- [ ] Set up local development environment
- [ ] Run the platform locally
- [ ] Complete first small task (bug fix or documentation)
- [ ] Submit first pull request

### Day 3: Customer Understanding

**Morning**
- [ ] Review customer personas and use cases
- [ ] Read customer success stories
- [ ] Understand our competitive landscape
- [ ] Review pricing and business model

**Afternoon**
- [ ] Listen to customer support calls (recorded)
- [ ] Review common issues and feature requests
- [ ] Shadow a customer success call (if available)
- [ ] Document your observations

### Day 4: Team & Culture

**Morning**
- [ ] 1:1 with your manager (goals, expectations)
- [ ] Meet each team member individually
- [ ] Learn about team rituals: standups, retrospectives, planning
- [ ] Understand our values: Autonomy, Openness, Speed, Quality

**Afternoon**
- [ ] Contribute to team discussion
- [ ] Participate in code review
- [ ] Ask questions (no question is too basic!)
- [ ] Start onboarding project (assigned by manager)

### Day 5: First Week Review

**Morning**
- [ ] Present onboarding project to team
- [ ] Get feedback and celebrate wins
- [ ] Review first week goals and achievements
- [ ] Set goals for next week

**Afternoon**
- [ ] Update your onboarding checklist
- [ ] Book 1:1s with key stakeholders
